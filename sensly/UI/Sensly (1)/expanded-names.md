## Quick Test Framework

For each name, ask:

1. **Say it 5 times fast** - Does it flow?
2. **Spell it over phone** - Is it easy?
3. **Google it** - Is it taken?
4. **Imagine the pitch** - "This is [NAME]..." - Does it sound right?
5. **Parent reaction** - Would a stressed parent remember it?
6. **Investor reaction** - Does it sound scalable/fundable?
