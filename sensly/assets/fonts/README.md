# Font Assets

## OpenDyslexic (required for dyslexia mode)

**License:** SIL Open Font License (OFL) — free for any use including commercial

**Download:**
1. Go to https://opendyslexic.org/ or https://github.com/antijingoist/opendyslexic
2. Download the latest release
3. Place these files in this directory:
   - `OpenDyslexic-Regular.otf`
   - `OpenDyslexic-Bold.otf`

**Why OpenDyslexic:**
- Designed specifically for readers with dyslexia
- Heavy weighted bottoms on letters reduce visual rotation/flipping
- Unique letter shapes reduce confusion between similar characters (b/d, p/q)
- Research: NIH PMC5629233 shows improved reading rate for some dyslexic readers
- British Dyslexia Association recommends it alongside increased letter spacing

**Alternative:** If OpenDyslexic is not available, the app falls back to the system font
with increased letter spacing (0.5px) and line height (1.6×) which still provides
meaningful accessibility improvement.
